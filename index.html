Here are some of my projects

  • Alien Invasion Project

    Python

    This project entails a spaceship, which acts as the item that shoots bullets at the aliens. The aliens start at the top of the screen and make their way down and the objective of the game is to make sure all the aliens are shot before the aliens touch the bottom of the screen. Once you shoot all the aliens, the next level ensues, where the aliens move faster, making it harder to shoot.

  • Pokémon Clone Database

    Java

    This project is similar to the Pokémon database, called Pakudex. Pakudex is a database where users can store their mythical creatures (Pakuri). Pakudex also has a variety of features, such as adding, sorting, and even evolving their Pakuri. By gathering more Pakuri, users can showcase their Pakudex to see who has the best collection.

  • Blackjack Game

    Java

    This project is a clone of the casino game blackjack, where the user plays against the computer. The goal of the game is for the sum of the cards in your hand to be the closest to 21, without going over 21, in comparision to the computer. The user is given a starting card and is given the option to "hit" (receive another card) or "stand" (remain with the cards in ur hand). The game continues until the user decides to stop playing, in which they can exit the game.
